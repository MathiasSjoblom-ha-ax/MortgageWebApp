För att starta
Öppna upp i intellij
Gå under src -> main -> java -> MortgageWebApp
Höger clicka på MortgageWebApp och kör "run 'MortgageWebApp.main()'"
Öppna chrome (css:en är designat för chrome) och gå till "http://localhost:8080/customers"
